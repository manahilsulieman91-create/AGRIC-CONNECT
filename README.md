# AGRIC-CONNECT
🌱 Agric Connect Website  Agric Connect is a simple and responsive web project designed to connect farmers, vendors, and buyers.   It includes multiple pages such as Home, Marketplace, About, Login, Sign‑Up, and a Price Calculator, all styled with shared CSS and supported by product images for a clean user experience

📂 Project Structure
HTML/
  index.html              # Home page
  marketplace.html        # Marketplace page
  about.html              # About page
  contact.html            # Contact page
  price.html              # Price calculator page
  vendors.html            # Vendors listing page
  CSS/
    style.css             # Global styles
    marketplace.css       # Marketplace-specific styles
    about.css             # About page styles
    price.css             # Price calculator styles
  IMAGES/
    market.png
    corn.png
    mangoes.png
    coffee.png
    rice.png
    wheat.png
    ... (other product images)
    
🌐 Branches

HOME → Contains `index.html` and shared assets.  
MARKET→ Contains `marketplace.html`, marketplace styles, and product images.  
ABOUT → Contains `about.html` and related styles.  
LOGIN-IN → Contains `login.html`.  
SIGN-UP → Contains `signup.html`.  
PRICE-CALCULATOR → Contains `price.html` and `price.css`.
